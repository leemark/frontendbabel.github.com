  • Developing Cross-Browser Extensions

    What do we do if we have to search for something? Of course, we fire up our favorite search engine web site. It is quite hard to push yourself to use another a different search engine rather than the usual one, even if you know that the result would be better. To change this UX pattern I developed Likeastore Chrome Extension. It adds social part to your search flow by showing relevant information from the article you liked. Besides Chrome we support Firefox and Safari. Despite the platform difference all these extentions are built from the same codebase.
